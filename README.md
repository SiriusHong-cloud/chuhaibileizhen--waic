# 出海避雷针⚡ V5 - 跨境出海全链路AI顾问

跨境出海全链路AI合规顾问平台，面向中小跨境电商卖家、外贸新手、品牌出海企业。从"合规检测工具"升级为全链路AI跨境出海顾问，覆盖文化雷区检测、合规法规扫描、本土化改造方案、关税与物流估算、店铺AI实时体检、选品合规快检、外贸实时新闻聚合等14大功能模块。

## ✨ V5.0 新增特性

### 🆕 四大全新模块
- **🏪 店铺AI体检** - 一键跳转商家店铺，AI实时分析店铺健康度、商品抽检、Listing评估、定价分析、侵权风险
- **📦 选品快检** - 输入选品URL/关键词，一键检测禁售/认证/关税/文化适配/竞争格局/利润空间
- **📰 外贸新闻** - 实时新闻聚合，中英双语切换阅读，智能标签关联对应检测模块
- **📚 外贸学院** - 新手学习区：入门教程、术语词典、实操指南、合规专题、运营技巧

### 🎨 界面全面升级
- 精致现代商务风设计，深色/浅色主题一键切换
- 14大模块卡片式布局，视觉层级清晰
- 全新品牌色：信任蓝#1A56DB + 警示橙#F59E0B + 安全绿#10B981
- 全响应式设计，适配桌面端、平板、手机

### 🌐 全面双语支持
- 界面、报告、新闻均支持中英双语切换
- 默认跟随用户语言偏好

## 🚀 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动服务
```bash
python main.py
```

服务将在 `http://localhost:8000` 启动。

## 🎯 AI人设

**出海避雷针⚡** — 一位资深的跨境出海合规AI顾问：
- 专业但不冷冰冰，像经验丰富的跨境老炮在给新手提点
- 说话直接，不绕弯子，发现雷区必须严肃警告
- 善用真实案例说明问题，不说空话
- 对文化差异保持敬畏，永远不做"这不重要"的判断

**口头禅：**
- "这有个雷，踩了会很疼"
- "别急，先让我帮你扫一遍"
- "这个坑，之前有品牌踩过，代价是XXX"

## 📊 评分系统

每次检测后返回综合评分：
- **基础分100**，每个风险点扣5-20分，每个安全项加5-10分
- 雷达图6维度：文化适配、合规达标、品牌安全、文案本土化、视觉适配、支付物流
- 风险等级：🟢安全 / 🟡谨慎 / 🟠高危 / 🔴禁止
